# 🍎 S.N.E.P. (Smart Neural Elite Partner)

Твой личный AI-ассистент, который всегда под рукой, видит твой экран и помогает писать код быстрее.

## 🚀 Что это такое?
S.N.E.P. — это умный оверлей (виджет) для рабочего стола, вдохновленный эстетикой Apple (iOS 17, Dynamic Island, Dark Mode). Он живет в углу экрана поверх всех окон. По одному клику он анализирует то, что ты сейчас видишь в своей IDE или браузере, находит баги, пишет код и работает как твой личный Senior-наставник.

Забудь про постоянные переключения между редактором и браузером с ChatGPT. S.N.E.P. видит контекст сам.

## ✨ Главные фичи
- 👁 **Умный анализ экрана:** Кнопка "Анализ экрана" делает снимок твоего рабочего пространства (при этом умно игнорируя сам виджет) и тут же выдает ревью кода или дизайна.
- 🎨 **Apple HIG Дизайн:** Темная тема, полупрозрачность, закругленные углы, приятные шрифты (Segoe UI / San Francisco) и микро-анимации.
- 💻 **Два режима работы:** 
  - **Режим Кода:** Отвечает как суровый Senior-разработчик. Без воды, сразу выдает чистый код.
  - **Режим Чата:** Отвечает как маркетолог, продакт-менеджер и кофаундер. Обсуждает стратегию и идеи.
- 🔐 **Anti-NDA (Встроенная защита):** Скрипт автоматически вырезает из твоих запросов и кода секретные токены, пароли и ключи перед отправкой в нейросеть. Твои секреты в безопасности.
- ⚡️ **Быстрые шорткаты:** Жми `Escape`, чтобы мгновенно свернуть или развернуть виджет (режим "таблетки").
- 📡 **Google Gemini Powered:** Под капотом используются новейшие модели Gemini 2.5 Flash / 2.0 Flash через официальное API.

## 🛠 Как запустить?

1. Убедись, что у тебя установлен **Python 3.10** или выше.
2. Установи нужные библиотеки (запусти в терминале):
   ```bash
   pip install -r requirements.txt
   ```
3. Получи бесплатный API-ключ от Google на сайте [Google AI Studio](https://aistudio.google.com/apikey).
4. Открой файл `.env` в папке с проектом и вставь туда свой ключ:
   ```env
   GEMINI_API_KEY=твой_ключ_здесь
   ```
5. Запускай ассистента:
   ```bash
   python main_ui.py
   ```

## 📁 Структура файлов
- `main_ui.py` — Главный интерфейс (PyQt6). Вся красота, кнопки, логика интерфейса и анимации живут здесь.
- `ai_analyzer.py` — Мозг проекта. Общается с Gemini API, управляет промптами и историей диалога.
- `screen_capture.py` — Умный захват экрана (через библиотеку `mss`). Умеет делать скрины всего монитора или активного окна.
- `anti_nda.py` — Скрипт-цензор, построенный на регулярных выражениях. Ищет и прячет пароли.
- `skill_executor.py` — Модуль для выполнения локальных команд (задел под будущие фичи автоматизации).
- `100_tasks.md` — Внутренний бэклог стартапа для режима Чата.
